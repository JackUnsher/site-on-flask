О плагине

Cursor MCP Creator - плагин для Figma, созданный для интеграции с Cursor IDE и быстрого прототипирования компонентов.

Возможности:

  • Экспорт структуры дизайна в Cursor IDE
  • Экспорт выбранных элементов как изображений
  • Создание компонентов с автоматической интеграцией в код
  • Выбор между светлой и темной темой
  • Различные варианты отображения изображений

Для работы плагина требуется запущенный WebSocket сервер Cursor IDE.
